# SaltedFish

咸鱼工具箱

1. ionic + angular 构建 Android APP

2. 实用小工具合集

主要用于熟悉 ionic。

## 已实现功能

1. 内置浏览器：必应搜索，常用网址导航

2. 扫一扫：扫描二维码，并解析二维码内容

3. 生成二维码: 把输入的文字转换成二维码
